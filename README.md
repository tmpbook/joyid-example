## Try it

## 1. Prepare a joy id in TestNet

Create one from here: [https://testnet.joy.dev](https://testnet.joyid.dev/)

## 2. Setup Project

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```
